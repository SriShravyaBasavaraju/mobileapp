Application name: No-more-nagging-inator

The App is designed for the moms to assign tasks to the children to avoid nagging behind them to complete their chores.

The startup screen shows a description of App and the way to use it. There is a navigation bar in the bottom of every screen using which the user can navigate from one screen to another.

The 3 different tasks the user can do are as follows:

1. Self- destruct -> Uninstalls the app
2. Children list  -> Displays a list of children. New children can be added or deleted from the same screen.
3. Chores list    -> Displays a list of chores to which the children are assigned. New chores can be added, assigned, deleted from here.

The user also has the ability to go back to the Home screen at any point of time using the Home button in the tool bar.
